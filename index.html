Om

Mitt namn är Jens Lundeqvist och jag bor i Östersund och jobbar som frisör till vardags. Jag har examen i datateknik från Mittuniversitet och en ekonomiutbildning som ekonomi/löneassistent från Avans.

Min kandidatexamen från Mittuniversitet tog jag 2005. Utbildningen hette Programvaruteknik och är inriktad mot mjukvaruutveckling och programmering vilket jag tycker är kul och spännande. Programmeringsspråk jag behärskar är Java, Javascript, C#, C++, PHP, TypeScript och lite Python.

2019 läste jag till ekonomi/löneassistent hos Avans där fick vi både jobba praktiskt och teoretisk lära oss företagsekonomi. Vi lärde oss använda program så som Fortnox och Visma men också Officepaketet. Praktik ingick också i utbildningen vilken jag tillbringade hos a2 revision i Östersund.

Våren 2020 läste jag en distanskurs i programmering mot Luleå Tekniska Universitet. Kursen heter Introduktion till programmering och C#. Jag tänkte läsa del 2 nu men den blev tyvärr inställd.

Länkar

Förmånsbeskattning av bil: En sida jag gjorde för att lära mig programmerings biblioteket React. Sidan hämtar data från skatteverkets api för bilförmån. Här kan man söka bland olika bilfabrikat och deras koder som används tillsammans med registreringsnummer för avdrag på lön för bilförmån. Programmeringsverktyg: Atom, React, Javascript, JSX, XML, GIT och Skatteverkets API

Bokföring av lön: En sida i Javascript som visar hur tabeller för preliminär skatt och skattetabell för löntagare. Sidan försöker visa hur dessa 2 tabeller hänger ihop. Finns också ett formulär som visar hur man bokför skatt och arbetsgivaravgift för anställda. Programmeringsverktyg: Atom, Javascript, HTML, JSON, GIT och Skatteverkets API

Bokföra-dagskassor: Denna sida är gjord i Javascript och jag använder den själv när jag bokför dagskassor. Eftersom vi säljer både produkter och tjänster ska dettaa bokföras på olika konton. Dessutom tar vi betalt både kontanter och bankinsättning via Swish och denna sida visar då vilka konton som ska bokföras. Programmeringsverktyg: VS Code, Javascript och HTML

Slå ihop swish-transaktioner konto hos Swedbank: Eftersom vi använder Swish och har företagskonto hos Swedbank så saknade jag ett sätt att lätt slå ihop Swish Transaktioner för dagar för att lättare stämma av bank konto 1930. Det Löste jag med Javascript och denna sida som kan omvandla ett företagskonto utdrag hämtat Swedbank i .xls format och sedan visa sammanslagna Swish inbetalningar per dag. Programmeringsverktyg: VS Code, Javascript, HTML och GIT

Enkel räknare i React: En enkel räknare jag kodade i React mest för att se om jag kunde lägga räknaren/appen i en undermapp på webbservern. Programmeringsverktyg: Atom, React, Javascript, JSX

Bokningsprogram i Java: Detta Java program gjorde jag redan 2009 och använder det fortfarande idag. Har även programmerat en webb bokningsmodul i PHP och mySQL så kunder kan boka vi Internet. Programmeringsverktyg Klient: NetBeans, Java och Derby databas. Server: NetBeans, PHP, HTML och MySQL databas.
