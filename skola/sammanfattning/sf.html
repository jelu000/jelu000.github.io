Variabler

Variabler i Javascrip deklarerar man med nyckel orden const, let eller var där const deklarerar en konstant variabel och let och var är variabler som kan ändras. Skillnaden mellan var och let är att var är ett gammalt sätt att deklarera variabler där variabeln blir global i filen och kan lätt leda till BUGGAR. En let variabel existerar bara i blocket där den deklareras.

Variabel typer

  • String
  • Number
  • Bigint
  • Null
  • Boolean
  • Undefined
  • Symbol
  • Object

Variabel Object-typer

  • Object
  • Array
  • Date

Ex:
const mitt_namn = "Jens" //Den här är av typen String
let my_age = 57 //Den här är av typen Number
let mina_djur = ["Hund", "Katt"]

Några Operatorer

  • +   &sn Addition
  • -   Subtraction
  • *   Multiplication
  • /   Division
  • %   Modulus (Division Remainder)
  • ++   Increment
  • --   Decrement

Några Tilldelnings Operatorer

  • = x   x = y
  • += x   x = x + y
  • -= x   = x - y

Några Jämförelse Operatorer

  • ==   equal to
  • ===   equal value and equal type
  • !=   not equal
  • !==  not equal value or not equal type
  • >  greater than
  • <  less than
  • >=  greater than or equal to
  • <=  less than or equal to

Några Logiska Operatorer

  • &&  logical and
  • ||  logical or
  • !  logical not

Några Typ Operatorer

  • typeof  Returns the type of a variable
  • instanceof  Returns true if an object is an instance of an object type
